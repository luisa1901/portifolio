PA e PG

Introdução ao tema

A progressão aritmética e progressão geográfica se referem com as sequências numéricas, que por sua vez se referem a sequências de números que seguem um padrão pré determinado. Elas podem ser finitas ou infinitas, a matemática foca em estudar as sequências que são possíveis fazer uma lei de formação.

Progressão aritmética

A progressão aritmética é a sequência na qual cada termo é o termo anterior somado com uma constante “r” (a partir do segundo). Essa constante é a diferença entre cada número da sequência. Por exemplo: se você subtrair o segundo número pelo primeiro você encontra a constante r.

Pode ser classificada em três categorias, crescente, decrescente e constante.

  • r > 0 crescente
  • r menor que 0 decrescente
  • r = 0 constante

Existe uma fórmula geral para encontrar um determinado número em uma PA

  • an = a1 + (n-1)r

an -> o termo geral

a1 -> o primeiro termo da PA

n -> número de termos ou a posição do termo a ser encontrado

r -> razão (constante r)

Progressão geométrica

Uma progressão geométrica é uma sequência numérica onde o termo anterior multiplicado por uma constante “q”, resultará no termo seguinte. Você pode encontrar a constante “q” a partir da divisão de de dois números da sequência. Por exemplo: dividindo o segundo número pelo primeiro se encontrará a constante “q”. Como a PA pode ser classificado em 3 categorias:

  • Crescente: PG de termos positivos: q > 1 PG de termos negativos: 0 < q < 1
  • Decrescente: PG de termos positivos: 0 < q < 1 PG de termos negativos: q> 1
  • Constante PG com todos os termos nulos: a1 = 0 e q qualquer numero PG com termos iguais e não nulos: a ≠ 0 e q = 1

Assim como a PA também existem uma fórmula geral:

  • an = a1 * qn-1

an -> o termo geral

a1 -> o primeiro termo da PG

n -> número de termos ou a posição do termo a ser encontrado

q -> razão (constante q)

Interpolação

  • Aritmética: a interpolação aritmética consiste em inserir um número x de números dentro de uma PA. Para isso nos informam sempre o a1 (termo inicial) e o an (o outro extremo), o principal objetivo da interpolação é descobrir a constante “r”, para assim dar a continuidade a sequência. Para isso utilizaremos a fórmula geral da PA, para descobrir o “n” é só somar o número de elementos a serem interpolados + 2 (termo inicial e final). Descobrindo o “n” já temos todas as informações para completar a equação e assim descobrir o “r” e realizar a interpolação.
  • Geométrica: a interpolação geométrica segue o mesmo propósito, inserir termo dentro de uma PG. No geral todos os passos são os mesmos, a única diferença é que depois de descobrir o “n” deve-se utilizar a fórmula geral da PG.

Exercícios:

Agora vou explicar um pouco sobre dois exercícios, um de PA e outro de PG. O primeiro que escolhi foi a questão 1 do terceiro trabalho. Resolvi escolher essa, pois é fácil de resolver e utiliza uma das fórmulas.

  1. A questão pede para calcular os 50 primeiros termos da PA, para isso temos a fórmula mencionada anteriormente. Porém para conseguir utilizar a fórmula da soma precisamos primeiro encontrar o “an” que no caso seria o 50o número, pois precisamos calcular a soma dos primeiros 50 números. Para isso devemos usar a fórmula geral an = a1 + (n-1)r.
  2. Para descobrir a “r” é só diminuir o segundo número pelo primeiro, ou seja, 5-1=4. Sendo assim fica: a50 = 1 + (50-1)*4. Depois de realizar as operações conseguimos descobrir que o 50o número é: a50 = 197.
  3. Agora o último passo é apenas substituir os dados na fórmula da soma: S50= ((1+197)*50) / 2. Realizando as operações chegamos ao resultado: S50 = 4950.

A próxima questão é a número 3 do terceiro trabalho:

  1. Para resolver essa questão é preciso inicialmente organizar esses dados. Pensando da forma que a1 = x + 1, a2 = x + 11, a3 = x + 41.
  2. Portanto, para conseguirmos achar o x podemos usar uma das relações, que diz que o “q” é igual ao a2/a1 que também é igual a a3/a2. Sendo assim temos que (x+11)/(x+1) = (x+41)/(x+11). A partir de então é só realizar multiplicação cruzada, obtendo a equação: (x+11)2 = (x+1)*(x+41).
  3. Realizando as operações, chegamos a conclusão de que x = 4. OBS: eu errei uma das potencia nessa questão por isso a resolução deu diferente.
